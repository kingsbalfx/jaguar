## SYSTEM OVERVIEW

### STEP 1: DAILY + 4HR BRIEF CONTEXT (2-3 MIN)

**Purpose:** Determine if the market setup is worth analyzing in detail

This is your **FILTER** - eliminates bad setups in 2-3 minutes before you waste time on H1→M15→M5→M1.

#### A. 24HR (Daily) Chart - 1 minute

**What to analyze:**
```
✓ Macro Structure:   Are there higher highs/lows (up) or lower highs/lows (down)?
✓ Liquidity Levels:  Where has price swept recently?
✓ FVGs:             Any Fair Value Gaps that need filling?
✓ Order Blocks:     From recent major moves?
✓ Volume Balance:   Is volume supporting the direction?
✓ Price Action:     Any rejection candles at key levels?
```

**If Daily is messy/consolidated:** ❌ SKIP
**If Daily is clear & directional:** ✅ CONTINUE to 4HR

---

#### B. 4HR Chart - 1-2 minutes

**What to analyze:**
```
✓ Recent Swings:    Latest highs and lows
✓ Trend Alignment:  Does 4HR agree with Daily bias?
✓ FVG Zones:       Which FVGs might get filled today?
✓ Order Blocks:    From which zones is price likely bouncing?
✓ Liquidity Events: Where have prices swept? (likely to return)
✓ Volume:          Is volume pushing price directionally?
✓ Price Action:    Any rejection candles visible?
```

**BUILD A LIQUIDITY MAP:**
- Where will price likely sweep?
- Where are FVG zones that need filling?
- Where are order blocks from recent moves?

**Decision:**
- Daily + 4HR **AGREE** (both up or both down)? → ✅ **CONTINUE to H1**
- Daily + 4HR **DISAGREE** (mixed signals)? → ❌ **SKIP**
- 4HR is **CONSOLIDATING/CHOPPY**? → ❌ **SKIP**

---

### STEP 2: H1 ANALYSIS (2-3 MIN)

**Purpose:** Confirm that H1 aligns with Daily + 4HR context

**Quick checks ONLY:**
```
✓ H1 Bias: Does it agree with Daily + 4HR?
✓ Liquidity: Any H1-level FVGs or sweep zones?
✓ Order Blocks: Where has H1 bounced from?
✓ Price Action: Any rejection candles?
```

**Decision:**
- H1 **ALIGNS** with Daily + 4HR? → ✅ **CONTINUE to M15**
- H1 **DISAGREES**? → ❌ **SKIP**

---

### STEP 3: M15 ANALYSIS (8-10 MIN)

**Purpose:** Find the EXACT setup and entry zone

**This is where 80% of your real work happens.**

**Deep analysis:**
```
✓ Swing Structure: HH/HL (up) or LL/LH (down)?
✓ Trend: Is it accelerating, decelerating, or consolidating?
✓ Liquidity Zones: Where will M15 sweep next?
✓ FVG Zones: Which FVGs are relevant today?
✓ Order Blocks: Where are the M15-level OBs?
✓ Volume: Any extreme imbalance on M15?
✓ Momentum: Are there momentum candles?
✓ Next Logical Move: Where should price move next?
```

**Your goal:** Identify:
1. The next swing high or low
2. The exact zone where it will form
3. The FVG(s) or OB(s) at that zone
4. The entry zone for M5 confirmation

---

### STEP 4: M5 CONFIRMATION (2-3 MIN)

**Purpose:** Wait for the setup to form exactly where you predicted

**Checks:**
```
✓ Rejection Candle: Is there a pin bar or reversal candle at expected zone?
✓ FVG Confirmation: Any M5-level FVG confirming the zone?
✓ Volume: Momentum on the rejection candle?
```

**Your job:** Don't force entries - wait for M5 to show the signal at the M15-predicted zone.

---

### STEP 5: M1 EXECUTION (1 MIN)

**Purpose:** Take the position when confirmed

**Execute when:**
```
✓ Break above/below the entry zone
✓ Momentum confirmed on break candle
✓ Good fill opportunity
```

**Trade management:**
- SL: Below the M5 setup zone (2-5 pips buffer)
- TP: At Daily/4HR reference levels (daily high/low, swing level)
